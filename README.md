# Smart Ski Rent

System do zarządzania wypożyczalnią sprzętu narciarskiego.

## Technologie
- Backend: Java, Spring Boot, Hibernate, MySQL
- Frontend: React, HTML, CSS
- Inne: Git, GitHub
